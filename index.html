Maya Control aims to provide an identical replica of https://packagecontrol.io - along with the surrounding infrastructure and tools - for the Autodesk Maya community.

Objectives

  • Encourage distribution of small Maya utilities
  • Encourage sharing and an open development of Maya utilities
  • Encourage automated testing for robust Maya utilities

Requirements

  • Packages can have no dependencies
  • Packages must each have a GitHub repository
  • Packages must be well documented, with (1) usage instructions, (2) screenshot/gif and (3) documentation
